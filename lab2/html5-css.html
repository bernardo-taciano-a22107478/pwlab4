What is HTML 5?

HTML stands for Hyper Text Markup Language. It is used to design web pages using a markup language.

Semantic Element Used Description
< header > Used on all pages at the top of each page.
< nav > Used on all pages at the top of each page to redirect to other pages.
< main > Used on all pages to include major sections within this element.
< article > Not Used.
< section > Used on all pages in order to separate the various contents addressed on each page.
< aside > Not Used.
< footer > Used on all pages at the bottom to reference the author of the project.
< details > Used on the local.html below the map, to reference a small interesting but not very important fact.
< meter > Not Used.
< progress > Not Used.
< abbr > Used in the html5-css.html page to abbreviate the word HTML.
< address > Implemented on all pages in the < footer > tag to reference the author's contact.

What is CSS?

CSS stands for Cascading Style Sheets. It's a style language used to describe the presentation of a document written in HTML

Selectors Used Description
*{ } Universal Used in css file, to change font family and center align text.
h1{ } Type Used in css file, to refer to titles h1, footer, table, nav, body, img, input.
.class{ } Class Used in the css file, to refer to specific classes.
#id{ } Id Very specific element, there can only be one and it was used to refer to each of the fieldsets in order to assign different background colors to them.
ol[type="intro-description"] Attribute Specifically used to give background color to ordered list elements with type="intro-description".
.nav:link{ } Pseudo-Class Used to assign characteristics to navigation menu links, to assign colors, before, during and after they are clicked.
p::first-letter Used to assign characteristics to the first letter of the paragraph element.
